VRM & Animation Resources for SpecsAI
=====================================

INSTALLED RESOURCES
-------------------
1. **Sample Model**: `assets/vrm/models/Alicia.vrm`
   - This is the standard "Alicia Solid" model.
   - You can use it immediately.

2. **Built-in Animations**:
   - The system now supports procedural animations without files:
     * "Nod" (Head up/down)
     * "Shake" (Head left/right)
     * "Jump" (Small hop)
   - Try saying: "Can you nod?" or "Jump for me!"

GETTING MORE (Wave, Dance, etc.)
--------------------------------
For complex moves, you still need to download files.

1. Free VRM Models (Characters)
--------------------------------
* **VRoid Hub**: https://hub.vroid.com/
* **Booth**: https://booth.pm/en/search/VRM (Search "VRM Free")

2. Free Animations (.glb / .gltf)
---------------------------------
* **Mixamo**: https://www.mixamo.com/
  - Download as FBX -> Convert to GLB -> Put in `assets/vrm/motions/`
  
Folder Structure
----------------
* Models: `assets/vrm/models/`
* Animations: `assets/vrm/motions/`
